    cd

    cp

    > Perintah 'cp' duntuk menyalin (copy) file atau direktori dari suatu lokasi ke lokasi lainnya.

    exit

    > Perintah untuk keluar dari shell.

    ls

    > ls - adalah perintah untuk melihat isi dari direktori `saat ini`

    mkdir

    > Perintah 'mkdir' digunakan untuk membuat sebuah direktori baru.

    mv

    > Perintah 'mv' digunakan untuk memindahkan atau menyalin file dan direktori dari satu lokasi ke lokasi lain.

    passwd

    > perintah **passwd** digunakan untuk mengatur password dari user yang sudah ada di sistem.

    pwd

    > untuk menampilkan direktori saat ini (current working directory).

    > Perintah 'rm' digunakan untuk menghapus file atau direktori pada sistem operasi Unix dan Linux.
